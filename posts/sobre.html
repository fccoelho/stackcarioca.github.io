Stack carioca é um projeto para listar ferramentas, módulos e aplicações com código aberto feitas por pessoas da comunidade Python do Rio de Janeiro.

O objetivo desse projeto é levar mais pessoas a contribuírem com software livre podendo programar com pessoas que já conheça ou possa conhecer em qualquer lugar.

Se você tem ou sabe de alguém que tenha um projeto Python com ligações com o Rio de Janeiro, crie uma issue ou envie um pull request para o nosso repositório.
