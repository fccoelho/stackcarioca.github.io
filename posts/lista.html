Publicado: 2016-06-12 17:38 [Código]

Aqui estão os projetos python feitos por pessoas da comunidade Python do Rio de Janeiro. Venha contribuir!

  • cloudoo Servidor de conversão de documentos XML-RPC

  • coopy Biblioteca simples, transparente e não intrusiva para persistência de objetos python.

  • feedbloom Leitor de feeds baseado em tarjas de canais de notícias.

  • fixofx Utilizatário python para arquivos de dados fincanceiros.

  • django-aggregate-if Agregações condicionais na queries Django.

  • django-test-without-migrations Rode testes do django sem necessidade de rodar migrações.

  • jsonbender DSL embarcada para transformação de dicionários.

  • model-mommy Factory de objetos para django.

  • nose-ipdb Rode testes com ponto de paradas do ipdb ao invés de pdb.

  • pypln-web e pypln-backend Pipeline distribuída para processamento de linguagem natural.

  • python-decouple Ferramenta para organizar configurações de projeto.

  • rows Uma interface comum e bonita para dados tabulares em vários formatos.

  • splinter Ferramenta para testar aplicações web com busca fácil por elementos, form actions e outras ações do browser.

  • sqlformatter Formatação de statements SQL com cores.

  • tenjinserver Ferramenta para facilitar a criação de templates tenjin
